# Customer Behaviour Analysis

## 📊 Project Overview

This project focuses on analyzing customer shopping behavior to generate actionable insights that can support business decision-making. The analysis combines SQL, Python, and Power BI to explore purchasing patterns, customer segments, and sales trends.

---

## 🎯 Business Problem

Businesses often struggle to understand customer purchasing behavior and identify high-value customers. This project aims to:

* Analyze customer transactions
* Identify key revenue drivers
* Support data-driven marketing and sales strategies

---

## 🛠️ Tools & Technologies

* SQL – Data extraction and querying
* Python – Data cleaning, analysis, and visualization
* Power BI – Interactive dashboard and reporting
* Excel – Initial data handling

---

## 📂 Dataset Description

The dataset contains customer shopping information, including:

* Customer demographics
* Purchase history
* Product categories
* Transaction details

---

## 🔍 Analysis Performed

* Customer segmentation analysis
* Revenue and sales trend analysis
* Category-wise performance evaluation
* Customer purchase frequency analysis
* Identification of top customers and products

---

## 📊 Dashboard


The Power BI dashboard provides:

* Sales overview by category
* Customer segmentation insights
* Monthly sales trends
* Top-performing products

---

## 💡 Key Insights

* A small group of customers contributes significantly to total revenue
* Certain product categories dominate sales performance
* Repeat customers play a crucial role in business growth
* Seasonal trends influence purchasing behavior

---

## 📁 Project Structure

* `data/` → Raw dataset
* `notebooks/` → Python analysis notebook
* `sql/` → SQL queries
* `powerbi/` → Power BI dashboard file
* `images/` → Dashboard screenshots

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the Jupyter Notebook for analysis
3. Run SQL queries in your preferred database environment
4. Open the Power BI file to explore the dashboard
